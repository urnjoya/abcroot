# abcroot

**abcroot** is a free online platform offering a variety of utility tools for text, developer, security, and general-purpose tasks. All tools are browser-based, fast, and privacy-friendly — no user data is stored.

---

## Live Website

[https://www.abcroot.xyz](https://www.abcroot.xyz)

---

## Features

- Convert text between uppercase, lowercase, title case, sentence case, and more.
- Perform find & replace, shuffle, sort, clean, and analyze text.
- Developer utilities like number conversion and markdown/text conversion.
- Security tools such as password generation.
- Utility tools including random color generation and text calculators.
- Fully responsive and lightweight; works in any modern browser.
- No server-side processing; everything runs locally in your browser.

## How to Use

1. Open [abcroot](https://www.abcroot.xyz) in your browser.
2. Navigate to the tool category you want to use (Text, Developer, Security, Utility).
3. Enter your text, number, or value as required by the tool.
4. Click the relevant button to perform the action (e.g., convert case, generate password, calculate).
5. Copy or download the output as needed.

All operations are performed locally, keeping your data private.
